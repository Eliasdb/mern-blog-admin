# Blog admin panel

## About

This is a MERN stack application through which users are able to:

- Create an account and login
- Create and update blog posts per user
- View all blog posts

## Used tech

- React
- React Quill
- MongoDB (Atlas)
- Express
- JWT
- Mongoose

## Script to run locally

Front end:
`cd client`
`npm i && npm run dev`

Backend:
`cd api`
`npm i && npx nodemon index.js`
